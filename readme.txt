"Autocomplete Using Binary Search Trees"


This program implements an auto complete functionality using Binary Search Trees. Given a file of names and weights it creates a Binary Search Tree.


To run this program:
gcc bst.c main.c -o myProg
./myProg


The following commands are supported:

1. stats (Given info on the Binary Search Tree)
2. find phrase (Finds the weight of a specific phrase)
3. add weight phrase (Adds a phrase with the weight given into the tree)
4. suggest k phrase (Given the user k options for a partial string entered)
5. exit (Quits the program)


Sample User Input

cities-by-population (or any other file in Data Files Folder)
Commands listed above after the "Ready" message.
